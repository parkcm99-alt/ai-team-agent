# AI Team Agent System

AI 팀 에이전트 시스템의 초기 프로젝트 구조입니다. 첫 버전은 구현보다 역할 분리, 문서화 위치, 테스트/예시 자료를 담을 공간을 명확히 만드는 데 집중합니다.

## 언어 정책

- 내부 명령어, 파일 이름, 시스템 프롬프트는 영어로 작성합니다.
- 사용자에게 보이는 출력, 보고서, 요약, 알림, 설명은 한국어로 작성합니다.
- 외부 API는 아직 연결하지 않습니다.

## 현재 범위

- 에이전트별 디렉터리와 기본 시스템 프롬프트를 준비했습니다.
- 템플릿, 하네스 테스트 케이스, 골든 예시, 실패 케이스, 문서 디렉터리를 준비했습니다.
- Google Sheets, Gmail, Slack, Telegram, Instagram, reporting 연동은 TODO로만 남겨두었습니다.

## 에이전트 역할

- `supervisor`: 작업 분배, 정책 적용, 결과 통합을 담당합니다.
- `assistant`: 일반 지원, 사용자 요청 정리, 간단한 산출물 초안을 담당합니다.
- `document`: 문서 생성, 요약, 구조화된 문서 산출을 담당합니다.
- `communication`: 이메일, 메신저, 알림 등 커뮤니케이션 흐름을 담당할 예정입니다.
- `workflow`: 반복 업무, 상태 전이, 작업 순서를 관리할 예정입니다.
- `harness`: 테스트 실행, 평가 기준, 회귀 검증을 담당할 예정입니다.
- `instagram`: Instagram 관련 콘텐츠 준비와 게시 흐름을 담당할 예정입니다.

## 하네스 통합 점검

하네스 정책, 리플레이 데이터셋, 문서 검증, 문서 워크플로, 커뮤니케이션 검증, Python 문법 검사를 한 번에 실행하려면 다음 명령어를 사용합니다.

```bash
python3 tests/harness/run_all.py
```

개별 정책 체커만 실행하려면 다음 명령어를 사용합니다.

```bash
python3 tests/harness/policy_checker.py
```

통합 점검은 결과를 한국어로 출력합니다. 하위 점검 중 하나라도 실패하면 0이 아닌 종료 코드를 반환합니다.
