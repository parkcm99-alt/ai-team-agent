# Slack Status Draft Template

### 제목

Slack 상태 요약 초안

### 채널 유형

Slack

### 메시지 목적

일일 상태 보고 초안과 팀 공유용 상태 요약

### 초안 본문

- 전체 상태: 확인 필요
- 완료된 작업: 확인 필요
- 남은 리스크: 확인 필요
- 다음 추천 작업: 확인 필요
- 이 메시지는 초안이며 실제 Slack 알림은 발송하지 않았습니다.

### 승인 필요 여부

필요

### 발송 여부

발송하지 않음

### 확인이 필요한 부분

- 수신 채널: 확인 필요
- 승인 담당자: 확인 필요
- 공유 마감일: 확인 필요
- 외부 실행 여부: 확인 필요

### 위험 요소

- 승인 전 Slack 전송 금지
- 외부 API 연결 없음
