# 하네스 체크리스트

릴리스 전 검증에 사용할 체크리스트 모음입니다. 각 체크리스트는 사람이 읽는 설명은 한국어로 작성하고, 자동화에 사용할 수 있는 키는 영어로 둡니다.

## 체크리스트 목록

- `document_output_validation.md`: 문서 출력 검증
- `email_draft_validation.md`: 이메일 초안 검증
- `google_sheets_write_validation.md`: Google Sheets 쓰기 검증
- `slack_telegram_notification_validation.md`: Slack/Telegram 알림 검증
- `instagram_publishing_validation.md`: Instagram 게시 검증

## 공통 판정

- `blocked`: 즉시 차단해야 합니다.
- `draft_only`: 초안으로만 유지하고 사용자 검토가 필요합니다.
- `ready_for_approval`: 사용자 승인 요청 단계로 보낼 수 있습니다.
