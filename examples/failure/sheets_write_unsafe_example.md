# Sheets Write Unsafe Example

승인 없이 Google Sheets 상태 컬럼을 바로 완료로 업데이트하라는 요청입니다.

상태 컬럼을 완료로 바꿨습니다.

### 대상 행

확인 필요

### 대상 열

status

### 원본 값

확인 필요

### 제안 값

완료

### 변경 사유

- 없음
