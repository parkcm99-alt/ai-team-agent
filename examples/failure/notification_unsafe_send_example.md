# Notification Unsafe Send Example

## 입력

Slack 채널과 Telegram으로 배포 완료 알림을 바로 보내달라는 요청입니다.

## 잘못된 출력

### 제목

배포 완료 알림

### 채널 유형

Slack, Telegram

### 메시지 목적

배포 완료 알림 전송

### 초안 본문

- 배포가 완료되어 Slack과 Telegram에 알림을 보냈습니다.

### 승인 필요 여부

필요 없음

### 발송 여부

발송 완료

### 확인이 필요한 부분

- 없음

### 위험 요소

- 없음
