# Notification Draft

### 제목

Telegram 긴급 알림 승인 요청 초안

### 채널 유형

Telegram

### 메시지 목적

긴급 알림 초안과 승인 요청 초안

### 초안 본문

- 긴급도: Slack/Telegram 실제 전송은 아직 연결하지 않았습니다.
- 요청 내용: Slack 상태 요약 공유는 명시적 사용자 승인 필요입니다.
- 다음 조치: 사용자가 초안을 검토하고 승인 여부를 결정해야 합니다.
- 이 메시지는 승인 요청 초안이며 실제 Telegram 알림은 발송하지 않았습니다.

### 승인 필요 여부

필요

### 발송 여부

발송하지 않음

### 확인이 필요한 부분

- 수신자: 확인 필요
- 긴급도: 확인 필요
- 승인 담당자: 확인 필요
- 마감일: 확인 필요
- 외부 실행 여부: 확인 필요

### 위험 요소

- 승인 전 Telegram 전송 금지
- 외부 API 연결 없음
- 긴급도가 불명확하면 알림을 보내면 안 됩니다.
