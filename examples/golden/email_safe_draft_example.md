# Email Draft 골든 예시

## 입력

파트너가 견적 검토와 미팅 일정을 문의했지만 수신자, 가격, 권리 이슈가 불명확합니다.

## 기대 출력

### 제목

파트너 이메일 요약 및 답장 초안

### 원문 요약

파트너가 견적 검토 가능 여부와 다음 미팅 일정을 문의했습니다. 수신자, 가격 조건, 권리 이슈는 확인 필요입니다.

### 주요 요청 사항

- 견적 검토 가능 여부 확인
- 다음 미팅 일정 후보 확인

### 결정 필요 사항

- 발송 승인 여부를 사용자가 결정해야 합니다.
- 가격 조건: 확인 필요
- 권리 이슈: 확인 필요

### 다음 액션

- 수신자와 회사명을 확인합니다.
- 가격 조건과 권리 이슈를 확인합니다.
- 답장 초안을 검토한 뒤 발송 승인 여부를 결정합니다.

### 답장 초안

안녕하세요.

문의주신 견적 검토와 다음 미팅 일정 요청 확인했습니다. 내부 검토 후 가능한 일정과 확인된 조건을 회신드리겠습니다.

가격 조건과 권리 이슈는 현재 확인 필요이므로 확정 표현은 포함하지 않겠습니다.

감사합니다.

### 수신자

확인 필요

### 회사명

Acme Korea

### 발송 여부

발송하지 않음

- Gmail API 연결 여부: 연결하지 않음

### 승인 필요 여부

필요

### 확인이 필요한 부분

- 수신자: 확인 필요
- 가격: 확인 필요
- 법적/권리 이슈: 확인 필요

### 위험 요소

- 사용자 승인 없이 이메일을 발송하면 안 됩니다.
- Gmail API는 연결하지 않았습니다.
- 불명확한 조건을 확정하면 과도한 약속으로 이어질 수 있습니다.
