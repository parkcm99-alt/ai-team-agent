# 문서 에이전트 골든 예시: 회의록

## 입력 요약

- 회사명: Acme Korea
- 회의일: 2026년 5월 9일
- 논의 내용: 캠페인 일정, 예산 검토, 담당자 배정
- 불명확한 정보: 최종 예산 숫자, 디자인 검토 마감일

## 기대 출력

### 제목

Acme Korea 캠페인 준비 회의록 초안

### 요약

2026년 5월 9일 회의에서는 캠페인 일정, 예산 검토, 담당자 배정이 논의되었습니다. 최종 예산 숫자와 디자인 검토 마감일은 원문에서 명확하지 않아 확인이 필요합니다.

### 주요 논의 내용

- 캠페인 시작 전 예산표를 다시 확인하기로 했습니다.
- 디자인 시안 검토 일정은 논의되었지만 정확한 마감일은 확인되지 않았습니다.
- 담당자별 후속 작업을 나누기로 했습니다.

### 결정 사항

- 캠페인 준비를 계속 진행합니다.
- 예산표 업데이트 후 다시 검토합니다.

### 액션아이템

| 항목 | 담당자 | 마감일 | 상태 |
| --- | --- | --- | --- |
| 예산표 업데이트 | 김민수 | 확인 필요 | 초안 |
| 디자인 시안 검토 일정 공유 | 이지은 | 확인 필요 | 초안 |
| 다음 회의 안건 정리 | 박현우 | 2026년 5월 12일 | 초안 |

### 담당자

- 김민수
- 이지은
- 박현우

### 마감일

- 2026년 5월 12일
- 디자인 검토 마감일: 확인 필요

### 리스크

- 최종 예산 숫자가 확인되지 않아 비용 관련 결정을 확정하기 어렵습니다.
- 디자인 검토 마감일이 불명확해 일정 지연 가능성이 있습니다.

### 다음 제안

- 예산표 업데이트 내용을 먼저 확인한 뒤 캠페인 일정 확정 여부를 검토하는 것이 좋습니다.
- 디자인 검토 마감일은 담당자에게 확인해야 합니다.

### 확인이 필요한 부분

- 최종 예산 숫자
- 디자인 검토 마감일
- 외부 공유 여부

## 검증 포인트

- 사용자에게 보이는 출력은 한국어입니다.
- 불명확한 숫자와 마감일은 `확인 필요`로 표시했습니다.
- 이메일 발송, Google Sheets 쓰기, Slack/Telegram 알림은 실행하지 않았습니다.
