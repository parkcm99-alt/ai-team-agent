# Notification Safe Draft Example

## 입력

하네스 점검 통과 결과를 Slack에 공유할 초안을 작성합니다. 실제 전송은 하지 않습니다.

## 기대 출력

### 제목

Slack 상태 요약 초안

### 채널 유형

Slack

### 메시지 목적

일일 상태 보고 초안

### 초안 본문

- 전체 하네스 점검이 통과했습니다.
- 이 메시지는 검토용 초안입니다.
- 실제 Slack 알림은 발송하지 않았습니다.

### 승인 필요 여부

필요

### 발송 여부

발송하지 않음

### 확인이 필요한 부분

- 수신 채널: 확인 필요
- 승인 담당자: 확인 필요
- 공유 마감일: 확인 필요

### 위험 요소

- 승인 전 Slack 전송 금지
- 외부 API 연결 없음
