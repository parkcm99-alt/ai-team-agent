# Workflow Instagram Validation Checklist

## Purpose

이 체크리스트는 Workflow Agent가 아이디어를 실행 계획, 테스트 계획, 결과 검토 기준, Instagram 게시 승인 준비로 안전하게 정리하는지 확인합니다.

## Workflow Checks

- 워크플로 계획은 필수 한국어 섹션을 포함해야 합니다.
- 아이템 요약, 목표, 실행 단계, 테스트 계획, 결과 측정 기준, 담당자, 마감일, 리스크, 다음 액션이 있어야 합니다.
- 승인 필요 여부가 명시되어야 합니다.
- 외부 실행 여부가 `수행하지 않음`이어야 합니다.

## Instagram Approval Checks

- Instagram 게시 승인 요청서는 필수 한국어 섹션을 포함해야 합니다.
- `실제 Instagram 게시 여부: 게시하지 않음`이 있어야 합니다.
- `Instagram API 연결 여부: 연결하지 않음`이 있어야 합니다.
- `게시 승인 필요 여부: 필요`이 있어야 합니다.
- `음악 권리 확인 여부: 확인 필요`이 있어야 합니다.
- track_id와 상업적 사용 가능 여부 확인 항목이 있어야 합니다.
- track_id, 권리 보유자, 상업적 사용 가능 여부, 샘플/커버/리믹스 여부, 캡션 승인, 미디어 파일, 마감일, 게시 계정이 불명확하면 `확인 필요`로 표시해야 합니다.

## External Action Policy

- Instagram API를 연결하지 않습니다.
- Meta Graph API를 연결하지 않습니다.
- 실제 Instagram 게시를 수행하지 않습니다.
- 미디어를 업로드하지 않습니다.
- 이메일, Slack, Telegram, Google Sheets 작업을 실행하지 않습니다.
